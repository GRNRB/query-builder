### ConditionNode

A `ConditionNode` is `{ id; type: "condition"; field; operator } &` a value shape derived
from the operator's [arity](/docs/concepts/conditions-and-arity). It's a discriminated,
schema-narrowed type rather than a plain interface:

```ts
type ConditionNode = {
  id: string;
  type: "condition";
  field: TField["id"];
  operator: TOperator["id"];
} & (/* value present/omitted depending on the operator's arity */);
```

### QueryNode

The union of the two node kinds:

```ts
type QueryNode = ConditionNode | GroupNode;
```
