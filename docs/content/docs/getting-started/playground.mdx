## How it's built

A few things to notice in the demo:

- **Operators are filtered per field.** Selecting a field only offers the operators that
  field supports — driven entirely by the schema.
- **Value inputs follow arity.** Operators with `arity: "one"` show a text input; the
  `is_set` operator (`arity: "none"`) shows *(no value)*. See
  [Conditions & arity](/docs/concepts/conditions-and-arity).
- **Dragging uses `moveNode`.** Reordering within the same group has a subtle index
  adjustment — covered in the [drag & drop guide](/docs/guides/drag-and-drop).

Read the annotated walkthrough in [Examples → Basic playground](/docs/examples/basic-playground).
